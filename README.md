## 📖 Descripción
Aplicación web interactiva para el control de gastos personales,
desarrollada semana a semana aplicando los conceptos modernos de JavaScript ES2023.
Este proyecto forma parte del Bootcamp de JavaScript donde se exploran
las características más modernas del lenguaje, desde fundamentos
hasta patrones avanzados como iteradores y generadores.
