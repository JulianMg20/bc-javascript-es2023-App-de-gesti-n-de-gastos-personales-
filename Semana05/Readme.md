## 🎯 Descripción
Dashboard interactivo que analiza gastos personales usando métodos
avanzados de arrays ES2023. Muestra estadísticas, rankings,
agrupaciones por categoría y tendencias mensuales.

## 📊 Funcionalidades Implementadas

1. **Aplanar Items Anidados** → `flatMap()`
2. **Top 5 Gastos** → `toSorted()` + chaining
3. **Agrupar por Categoría** → `reduce()`
4. **Tendencia Mensual** → `reduce()` + `toSorted()`
5. **Último Gasto Alto** → `findLast()` ES2023
6. **Modificar sin Mutar** → `with()` ES2023
7. **Filtrar por Fecha** → `filter()` + chaining
8. **Categorías Únicas** → `Array.from()` + `Set`
9. **Estadísticas Generales** → `reduce()`
10. **Filtro Interactivo** → `toSorted()` + DOM
